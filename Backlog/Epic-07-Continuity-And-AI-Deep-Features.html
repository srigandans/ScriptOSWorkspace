Description
Two capabilities that mature ScriptOS into an intelligent series platform: (1) Continuity Service — SeriesTimeline graph tracking cross-episode story-day assertions, automatic violation detection with AI-assisted ambiguous case resolution, and a violation management workflow; (2) AI Deep Features — character voice registry completion with few-shot reference scenes, semantic cache (pgvector ANN at 0.97 threshold), advanced analytics (coverage_analysis, structure_analysis), C2PA manifest in PDF exports, and hybrid search hardening. Beta milestone at Sprint 11 end (Week 24): 3 production teams onboarded. Reference: Wiki/15-roadmap.md Sprints 11–12.

  • Beta milestone (Week 24): 3 production teams using breakdown + scheduling + on-set
  • Semantic cache: second identical request hits cache (model_id = 'semantic_cache', cost_usd = 0)
  • Voice violation warning shown when AI suggests a character's forbidden word
  • Coverage analysis generates structured report for 90-page test script
  • Hybrid search Recall@10 improved ≥ 15% after BM25/RRF tuning
