#199EpicProduction Planning — Scheduling, Budget & Publish Saga
Sprints 7–8 ● Priority: Critical schedulingbudgetpublish-sagatemporal
Description
Build the three production planning services that transform a completed script breakdown into an actionable shooting schedule, budget, and a locked production version: (1) Scheduling Service — OR-Tools constraint solver, stripboard management, DOOD tables; (2) Budget Service — budget lines, account codes, EP Budgeting XLSX export; (3) Publish-to-Production Saga — full seven-step Temporal workflow with compensation, 24h escalation timers, 48h auto-approve, and parallel/sequential approval routing. The Notification Service (SendGrid email + WebSocket push) is also delivered in this epic to support the approval workflow. Reference: Wiki/15-roadmap.md Sprints 7–8, Wiki/07-workflow-orchestration.md, Wiki/08-production-modules.md.
Acceptance Criteria
  • OR-Tools solver produces valid schedule for 30-scene test project in < 5s
  • Stripboard drag-and-drop works; manual overrides preserved across solver re-runs
  • DOOD table calculates W/H/T/F correctly for all characters
  • Full publish saga runs end-to-end: lock → validate → breakdown → schedule → budget → approval → lock
  • Approval email sent via SendGrid; in-app notification received within 2s
  • Compensation fires in correct reverse order on rejection
  • 48h auto-approve fires correctly in Temporal time-skipping integration test
  • Budget XLSX export validates against EP Budgeting account code structure
  • All services deployed to staging with passing health checks
