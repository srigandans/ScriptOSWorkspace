Description
Sprint 15 delivers the full Legal Service (replacing the Sprint 8 stub): rights tags, NDA gates, legal holds, and complete validateScriptPolicy. GDPR data export and account deletion. Sprint 16 is the security and accessibility hardening sprint: pentest remediation, Zod validation on all mutations, global API rate limiting, RBAC matrix automated tests, SOC 2 Type I evidence package, backup restoration verification, 500-user load test (P99 < 500ms), WCAG 2.2 AA audit and remediation, and Lighthouse score ≥ 90. This epic is the final gate before GA hardening. Reference: Wiki/15-roadmap.md Sprints 15–16, Wiki/11-security-compliance.md.

  • validateScriptPolicy blocks publish when rights are missing or NDA not signed
  • NDA gate: 403 returned for unsigned writers on NDA-gated scripts
  • Legal hold: DELETE returns 403 with hold reason in audit log
  • GDPR data export produces complete JSON package for a test user
  • AI ledger entries anonymized (not deleted) after account deletion — WGA compliance preserved
  • All pentest critical/high findings resolved
  • Load test: 500 concurrent users; P99 < 500ms; error rate < 0.1%
  • WCAG 2.2 AA audit passed; all Level AA criteria pass axe-core scan
  • SOC 2 Type I readiness package complete
